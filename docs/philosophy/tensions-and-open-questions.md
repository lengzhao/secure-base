# 张力与开放问题

> 概念阶段 deliberately 保留未决问题。P0 五题已闭合为 [决议草案](../decisions/2026-06-13-p0-mechanism-resolutions.md)，其余标出分叉点供后续讨论。

## 1. 已闭合（P0，2026-06-13）

| 问题 | 结论摘要 |
|------|----------|
| 必需品边界 | 三层清单（核心 / 扩展 / 非必需）；心理服务暂搁置 |
| Tier 0 | 接近均等；差距 ≤ 1.5 倍；占成员-facing 分配 ≥ 40% |
| Tier 1+ | 单周期加成 ≤ Tier 0 × 30%；贡献分中位数 ≤ 3 倍 |
| 不作恶清单 | 成文双清单；惩罚须事后、可申诉、可恢复 |
| 信用分 vs 贡献分 | 分离展示；会员费只影响 Tier 1+ |

细则见 [规则草案 v0.1](../drafts/rules-v0.1.md)。

## 2. 核心张力（仍开放）

| # | 张力 | 过松 / 过紧风险 | 当前倾向 |
|---|------|----------------|----------|
| 1 | 宽松 vs 防滥用 | 搭便车 ↔ 道德审判 | Tier 0 低而稳；作恶才除名 |
| 2 | 实名 vs 隐私 | 责任链 ↔ surveillance | 实名 + 聚合公开 + 脱敏 |
| 3 | 集体 vs 个人自由 | 集体决策 ↔ 退出权 | 可自由退出；规则明示 |
| 4 | AI 养人 vs 劳动尊严 | 价值感缺失 ↔ 强制劳动 | AI 51% 归集体；人定目标与仲裁 |
| 5 | 集体控制 vs 投资者 | 冷启动难 ↔ 资本吞没 | 51% 控方向；49% 有限回报 |
| 6 | 保障 vs 逆向选择 / 道德风险 | 吸引搭便车 ↔ 底太薄 | 观察期、共付、低而稳的 Tier 0 |
| 7 | 规模 vs 信任 | 熟人伦理 ↔ 陌生人制度 | 试点建议 300–500 人 |
| 8 | 农业 / 工业优先 vs 多元化 | 供给能力 ↔ 集中风险 | 以农业 / 基础工业为主 |
| 9 | 公开透明 vs 隐私 | 黑箱 ↔ 羞辱 | 系统性公开；个人脱敏 |
| 10 | 积分兑换 vs 外销增长 | 成员优先 ↔ 池子兑空 | 兑换优先履约 + 增长监控 |
| 11 | 道德追责 vs 道德泛化 | 规则可信 ↔ 生活方式审判 | 仅清单内作恶可扣分 |
| 12 | 与国家关系 | 互补 ↔ 合规风险 | 第三层安全网，非替代 |

**仍待定量**：Tier 0 额度「仅生存」还是「有尊严」？实名粒度？外销 / 兑换比例？风险准备金最低水位？
