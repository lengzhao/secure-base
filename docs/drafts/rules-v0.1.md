> 状态：草案 · 非执行  
> 依据：[P0 机制决议](../decisions/2026-06-13-p0-mechanism-resolutions.md)

P0 决议定原则与数值边界；本文档补充可执行细则。

---

## 1. Tier 0

**目的**：合格成员定期领取接近均等的积分，兑换核心必需清单。

### 资格

| 获得 | 失去（仲裁确认后） |
|------|-------------------|
| 实名 + 不作恶承诺 + 观察期（建议 1 月，期间 50% 积分） | 欺诈申领、倒卖集体物资、严重损害集体 |

**不影响 Tier 0**：会员费、贡献分、劳动多少、失业 / 疾病。

### 发放

| 项目 | 规则 |
|------|------|
| 周期 | 每月 1 次，固定日期 |
| 差距上限 | 最高 / 最低 ≤ 1.5 倍（仅观察期、地域系数） |
| 积分 | 不可转让、不可变现；未用可结转 1 周期 |
| 用途 | 仅兑换核心必需清单；兑换优先于外销排产 |

成员信用分仅做风控（正常 / 暂停 / 恢复），**不**改变 Tier 0 额度。

### 会员费关系

| 场景 | Tier 0 | Tier 1+ |
|------|--------|---------|
| 按时缴纳 | 正常发放 | +5 贡献权益分 / 月 |
| 未缴 | 正常发放 | 不加分；连续 6 月贡献分衰减 |

Tier 0 不是付费会员制。减免 / 缓缴 / 劳动替代通道暂不考虑。

### 试点参数（示例）

观察期 1 月；满资格 100 分 / 月；100 分 ≈ 5 kg 大米 + 0.5 L 油。数值试点前须重校。

## 2. 信义分

对外统称「信义分」，内部分拆为**成员信用分**与**贡献权益分**，独立计算、独立展示。

### 成员信用分（资格 / 风控）

| 事件 | 变化 |
|------|------|
| 实名 + 不作恶承诺 | 初始 100 |
| 连续 12 月无违规 | +10（上限 +200 奖励分） |
| 虚假申领 | -50 ~ -200，暂停 Tier 0 |
| 倒卖集体物资 | -100 ~ -300 |
| 严重损害集体 | -200 ~ 除名 |

信用分不因贡献多寡增减。申诉成功可恢复；赔偿 + 6 月无违规可申请部分恢复。

### 贡献权益分（Tier 1+ / 治理）

| 来源 | 分值 |
|------|------|
| 会员费按时缴纳 | +5 / 月 |
| 连续 12 月按时缴纳 | +10 一次性 |
| 劳动 / 知识 / 治理 / AI 协作 | +2 ~ +20 / 次 |
| 实物会员费缴付 | 按公开估值规则折算 |

超额缴纳会员费不额外加分。连续 6 月零贡献且未缴会员费：-2 / 月，不低于初始值。

**Tier 1+ 计算示例**：`Tier1_bonus = Tier0 × min(contrib_weight, 0.30)`

不作恶清单与争议流程见 [P0 §4](../decisions/2026-06-13-p0-mechanism-resolutions.md#4-不作恶底线清单)。扣分仅适用于清单内「属于作恶」的行为。

## 3. 会员费与资金

**会员费**是成员维护系统的稳定参与投入（非「月捐」、非慈善施舍），进入集体资金池，按时缴纳增加 Tier 1 贡献权益分，**不**增加 Tier 0 额度。

| 项目 | 示例 |
|------|------|
| 建议区间 | 50–200 元 / 月（中位数 100） |
| Tier 1 加分 | +5 / 月；连续 12 月 +10 一次性 |

### 池子分配顺序

1. 运维与 AI 成本 → 2. 风险准备金（补至最低水位）→ 3. Tier 0 → 4. Tier 1+ → 5. 实物储备 → 6. 再投资 / 应急

Phase 1 不做 51/49 投资者结构。

### 风险准备金

最低水位 3 个月运营 + Tier 0 支出；低于 2.5 月暂停 Tier 1+ 增量；低于 2 月公开预警。

### Phase 1 资产结构

实物储备 40% · 现金 / 货基 40% · 应急互助 20%

Phase 1 无外销，监控兑换消耗 vs 会员费流入；Phase 2+ 启用外销增长平衡（外销 ≥ 30% 可售产出、兑换履约 SLA 等）。

## 4. 透明治理

**原则**：规则、账本、资金、分配、惩罚案例原则上公开；隐私脱敏，不以隐私隐藏系统性风险。

| 频率 | 内容 |
|------|------|
| 每月 | 账本、会员费收缴率、Tier 0 发放与履约、风险准备金、AI 成本 |
| 每季度 | Tier 1+ 分布、投资标的与收益、审计摘要 |
| 事件驱动 | 超池子 5% 支出、投资失败、资金异常、AI 伦理事件 |

**Phase 1 最小机构**：成员代表大会（规则 / 预算）、理事会（执行）、资金管理小组、信义分仲裁庭；均设轮换。

**决策**：日常理事会执行；一般事项公示 7 天 + 简单多数；重大事项（章程、核心清单、预算大改）公示 14 天 + 2/3 多数。异议须登记、公开计数。

## 5. AI 边界（Phase 1）

**原则**：AI 产出归集体；敏感决策 human-in-the-loop；使用范围与成本公开。

| 允许 | 禁止 |
|------|------|
| 账本只读问答 | 自动扣分 / 除名 |
| 需求 - 库存匹配（人工确认后执行） | 自动资金划拨 |
| 月度报告草稿（人工审核后发布） | 成员健康 / 信用全自动评估 |
| | 用成员数据对外训练；全自动医疗 / 法律建议 |

Phase 1 AI 成本 ≤ 运维预算 10%。Phase 2+ 扩展（贡献自动登记、对外轻量服务）须成员审议通过。
