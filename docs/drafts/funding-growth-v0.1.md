# 资金与增长草案 v0.1

> 状态：草案 · 非执行  
> 依据：[P0 机制决议](../decisions/2026-06-13-p0-mechanism-resolutions.md)、[集体资金](../design/collective-fund.md)

## 1. 会员费定位

**会员费**（非「月捐」）是成员共同维护系统的**稳定参与投入**：

- 语义：共同体成员义务，非慈善捐赠
- 用途：进入集体资金池，支撑 Tier 0、储备、运营
- 激励：按时缴纳增加 **Tier 1 贡献权益分**，不增加 Tier 0 额度

---

## 2. 会员费规则（试点示例）

| 项目 | 示例 |
|------|------|
| 建议区间 | 50–200 元 / 月（自愿区间内承诺，中位数 100） |
| 缴纳周期 | 每月 |
| Tier 1 加分 | +5 贡献权益分 / 月 |
| 连续 12 月 | 额外 +10 一次性 |
| 未缴 | 不影响 Tier 0；不计 Tier 1 贡献权益分 |

---

## 3. 池子分配顺序

1. 运维与 AI 成本
2. 风险准备金（补至最低水位）
3. Tier 0 成员保障
4. Tier 1+ 贡献加权
5. 实物储备补充
6. 再投资 / 应急互助

**Phase 1 刻意不做**：51/49 投资者结构、二级市场投资。

---

## 4. 风险准备金

| 参数 | 建议值 |
|------|--------|
| 最低水位 | 3 个月运营 + Tier 0 支出 |
| 触发补库 | 低于 2.5 个月时，暂停 Tier 1+ 增量分配 |
| 触发预警 | 低于 2 个月时，成员公开预警 |

---

## 5. Phase 1 资产结构

| 篮子 | 比例 | 说明 |
|------|------|------|
| 实物储备 | 40% | 大米、食用油等 |
| 现金 / 货基 | 40% | 医疗互助、应急 |
| 应急互助池 | 20% | 突发支取 |

---

## 6. 积分兑换与外销增长（Phase 1）

Phase 1 无外销，仅监控**兑换消耗 vs 会员费流入**：

| 指标 | 目标 |
|------|------|
| 月会员费流入 | ≥ Tier 0 兑换成本 + 运维 |
| 储备覆盖 | ≥ 1 个月 Tier 0 实物兑换量 |
| 池子健康度 | 现金 + 储备 ≥ 2 个月支出 |

Phase 2 引入外销后，启用完整增长平衡指标（见下）。

---

## 7. 增长平衡指标（Phase 2+ 预备）

| 指标 | 建议方向 |
|------|----------|
| 外销占可售产出 | ≥ 30% |
| 成员积分兑换占可售产出 | ≤ 70% |
| 外销增量 / 积分兑换消耗 | ≥ 1.0（滚动 3 个月） |
| 兑换履约 SLA | 成员兑换优先于外销排产 |

---

## 8. 资金公开

每月公开：

- 会员费收缴总额、收缴率
- 各篮子余额与变动
- 风险准备金水位与覆盖率
- Tier 0 / Tier 1+ 支出占比
